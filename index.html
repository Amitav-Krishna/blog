Publications

Frobenius Normalization Enables Stable Training for Quantum State Denoising
Under review at IEEE QCCL 2026

A data preprocessing method that stabilizes neural network training by normalizing input distributions. Demonstrated scaling from small to large problem sizes (5-qubit to 8-qubit quantum systems) by addressing gradient suppression from scale mismatches.

Projects

Side projects exploring systems, ML infrastructure, and developer tools.

cerebras-challenge — LLM Token Visualizer

Developer tool for visualizing LLM token perplexity, suggesting alternate generation paths, and highlighting high-saliency tokens. Designed as a passive assistant rather than autonomous agent.

tinygrad-verilog — ML Compiler to Hardware

Started as Verilog neural network (XOR training), evolved into Python backend generating Verilog from tinygrad compute graphs. Devlog playlist.

llm-from-scratch — Neural Network Framework in C++

Autograd engine, math primitives, and neural network layers implemented from scratch in C++. Educational project exploring automatic differentiation and linear algebra foundations.

gpu_torrent — Distributed GPU Inference

BitTorrent-like protocol for distributed token generation across self-hosted GPUs. Built with FastAPI, Redis, PostgreSQL. Demo video.

stock_news — Full-Stack Stock Research

Next.js app displaying stock price timelines with overlaid news articles to identify patterns between coverage and price movement. PostgreSQL, yfinance, GNews API.

search_engine — Word2Vec from Scratch

Implementation of word embeddings following Jake Tae's tutorial. Skip-gram model with negative sampling trained on custom corpus.

face_recognition_ai — Face Matching with DeepFace

Face detection and recognition using DeepFace (Facenet) embeddings with cosine similarity matching. Demo video.

Stock Email Notifications — First Project (Grade 7)

Automated email alerts for stocks near 52-week highs/lows. Python, yfinance, smtplib, Google Sheets API. Reached ~80 users before shutting down.

eye-tracking — Gaze-Controlled Cursor

Python + OpenCV + dlib for webcam-based eye tracking to control mouse cursor. Demo video.
